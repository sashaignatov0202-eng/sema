# Soft Particles OpenGL Implementation

Реализация техники "Мягких частиц" (Soft Particles) на C++ и OpenGL (Modern GL 3.3+).
Проект демонстрирует решение проблемы резких границ при пересечении полупрозрачных спрайтов с геометрией сцены с использованием Depth Buffer.


## Управление
* WASD — Перемещение камеры
* Мышь — Обзор
* Space / Shift — Вверх / Вниз
* ESC — Выход

# Технологии
* C++17
* OpenGL 3.3
* GLFW — Оконный интерфейс и ввод
* GLAD — Загрузчик OpenGL
* GLM— Математическая библиотека
* stb_image — Загрузка текстур

# Как запустить (Build)
1. Убедитесь, что у вас установлена Visual Studio 2019/2022.
2. Установите/скачайте библиотеки GLFW** и GLM.
   * *Примечание: В проекте могут быть настроены абсолютные пути к библиотекам. Вам нужно будет указать пути к `include` и `lib` папкам GLFW/GLM в настройках проекта (Свойства -> C/C++ -> General / Linker).*
3. Откройте файл `sema.sln`.
4. Соберите проект (Build Solution) в конфигурации **x64**.
5. Убедитесь, что папки `shaders/` и `resources/` находятся рядом с `.exe` файлом или в рабочей директории отладки.
